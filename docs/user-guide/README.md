# User Guide

## Hướng dẫn sử dụng hệ thống

### 1. Upload văn bản
- Upload file PDF, DOCX, hoặc text
- Hệ thống sẽ tự động OCR và xử lý

### 2. Xử lý văn bản
- Chia văn bản theo cấu trúc pháp lý Việt Nam
- Trích xuất metadata tự động
- Phân loại bằng AI

### 3. Kết quả
- Nhận được các blocks với metadata chuẩn hóa
- Có thể export thành Markdown hoặc JSON

## Cấu trúc văn bản được hỗ trợ

- **Chương** → **Điều** → **Khoản** → **Điểm**
- **Căn cứ pháp lý**
- **Quy trình** với các bước
- **Pattern đặc biệt** như "như sau:"
